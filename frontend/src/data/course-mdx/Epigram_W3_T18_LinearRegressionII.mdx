# Regression Under Repeated and Correlated Data

::: {.workedbox number="1" difficulty="medium" problemid="voleon_group_qdsi_p4"}
**Effect of Duplicating Observations in OLS Regression.** What happens if you duplicate all $n$ data points (creating $2n$ observations) to: (i) coefficient estimates, (ii) standard errors and $t$-statistics, (iii) $R^2$, (iv) chi-square statistic, (v) reduced chi-square?
:::

::: solutionbox
Let $\hat{\beta}$, $\hat{\sigma}^2$, $R^2$ etc. denote the original quantities. After duplication ($n\to2n$):

**(i) Coefficient estimates $\hat{\beta}$: unchanged.** Doubling every observation is equivalent to weighting all observations equally by 2. The OLS formula $\hat{\beta}=(X^\top X)^{-1}X^\top y$ doubles both $X^\top X$ and $X^\top y$, and they cancel: $\hat{\beta}_{\text{dup}}=(2X^\top X)^{-1}(2X^\top y)=\hat{\beta}$.

**(ii) Standard errors: halved. $t$-statistics: multiplied by $\sqrt{2}$.** $\hat{\sigma}^2_{\text{dup}} = \text{RSS}_{\text{dup}}/(2n-p)
\approx 2\,\text{RSS}/(2n) = \hat{\sigma}^2$ (same variance estimate). $\mathop{\mathrm{Var}}(\hat{\beta}_{\text{dup}})=\hat{\sigma}^2(2X^\top X)^{-1}
= \frac{1}{2}\mathop{\mathrm{Var}}(\hat{\beta})$. Standard error $\to$ SE/$\sqrt{2}$. $t$-statistic $= \hat{\beta}/\text{SE}
\to \sqrt{2}\cdot t$.

**(iii) $R^2$: unchanged.** $R^2 = 1-\text{RSS}/\text{TSS}$. Both RSS and TSS double under duplication, so their ratio is preserved. $R^2$ is unchanged.

**(iv) Chi-square statistic $\chi^2 = \text{RSS}/\sigma^2$: doubled.** RSS doubles but $\sigma^2$ is fixed (assumed known), so $\chi^2$ doubles.

**(v) Reduced chi-square ($\chi^2_\nu = \chi^2/\nu$, where $\nu=n-p$): approximately unchanged.** $\chi^2_\nu = \text{RSS}/((n-p)\sigma^2)$. After duplication: $2\,\text{RSS}/((2n-p)\sigma^2)\approx\text{RSS}/((n-p)\sigma^2)$ for large $n$. It stays approximately equal to 1 if the model is correctly specified.

**Key takeaway:** Duplicating data *artificially inflates* precision (halves SE, doubles $t$). The model fit ($R^2$, coefficients) is unchanged --- your model is not better, you just pretend to have more information than you do. This is why independent observations are a core assumption of OLS.
:::

::: {.freeproblem number="2" difficulty="medium" problemId="citadel_qdsi_p6"}
**When Weak Predictors Combine into a Strong Linear Model.** Three separate regressions $Y=aX_1$, $Y=bX_2$, $Y=cX_3$ each have very low $R^2$. Can the multiple regression $Y=\beta_1 X_1+\beta_2 X_2
+\beta_3 X_3$ have a much higher $R^2$? Explain.
:::

::: solutionbox
**Yes, absolutely.** This phenomenon is well-illustrated by the following construction.

**Key insight: uncorrelated predictors combine additively.** If $X_1, X_2, X_3$ are mutually uncorrelated and $Y = X_1+X_2+X_3
+\varepsilon$, then:

- Each individual regression $Y\sim X_j$ has $R^2_j
          = \mathop{\mathrm{Var}}(X_j)/\mathop{\mathrm{Var}}(Y)$. If $\mathop{\mathrm{Var}}(X_j)$ is small relative to $\mathop{\mathrm{Var}}(Y)$, each individual $R^2$ is low.

- The multiple regression $Y\sim X_1+X_2+X_3$ has $R^2 = (\mathop{\mathrm{Var}}(X_1)+\mathop{\mathrm{Var}}(X_2)+\mathop{\mathrm{Var}}(X_3))/\mathop{\mathrm{Var}}(Y)$, which can be close to 1 even when each individual $R^2$ is small.

**Numerical example.** Let $Y=X_1+X_2+X_3+\varepsilon$ with $X_j\stackrel{\text{iid}}{\sim}N(0,1)$ and $\varepsilon\sim N(0,0.01)$. Then $\mathop{\mathrm{Var}}(Y)\approx3$.

- $R^2(Y\sim X_j) = 1/3 \approx 33\%$ each.

- $R^2(Y\sim X_1+X_2+X_3) = 3/3.01 \approx 99.7\%$.

In general: if $R^2_j = r$ for all $j$ and the $X_j$ are uncorrelated: $R^2_{\text{multiple}} = 3r/(1-\text{noise fraction})$.

**Why does this happen?** The column space of $(X_1, X_2, X_3)$ is much richer than any individual column. Even if each predictor explains a small fraction of $\mathop{\mathrm{Var}}(Y)$ on its own, together they can explain most of it --- provided they capture *different* aspects of $Y$ (i.e. they are not collinear).

**Key takeaway:** Low individual $R^2$ does not imply a low combined $R^2$. Uncorrelated predictors combine orthogonally, and their explanatory power adds up. This is a fundamental reason to include multiple predictors in a model even when each one individually appears weak.
:::

::: {.freeproblem number="3" difficulty="hard" problemId="citadel_qdsi_p1"}
**How Correlated Covariates Change Regression Coefficients.** Assume centered variables with $\mathop{\mathrm{Corr}}(Y,X_1)>0$, $\mathop{\mathrm{Corr}}(Y,X_2)=0$, and $\mathop{\mathrm{Corr}}(X_1,X_2)\neq0$. Compare the coefficient on $X_1$ in (1) $Y=aX_1$ and (2) $Y=b_1X_1+b_2X_2$. Which is larger?
:::

::: solutionbox
**Claim:** $b_1 > a > 0$.

**Derivation.** Simple regression: $a = \mathop{\mathrm{Cov}}(Y,X_1)/\mathop{\mathrm{Var}}(X_1) > 0$.

Multiple regression normal equations (centered variables): $$\begin{align*}
  b_1\mathop{\mathrm{Var}}(X_1) + b_2\mathop{\mathrm{Cov}}(X_1,X_2) &= \mathop{\mathrm{Cov}}(Y,X_1) \\
  b_1\mathop{\mathrm{Cov}}(X_1,X_2) + b_2\mathop{\mathrm{Var}}(X_2) &= \mathop{\mathrm{Cov}}(Y,X_2) = 0
\end{align*}$$ From the second equation: $b_2 = -b_1\mathop{\mathrm{Cov}}(X_1,X_2)/\mathop{\mathrm{Var}}(X_2)$.

Substitute into the first: $$b_1\mathop{\mathrm{Var}}(X_1) - b_1\frac{\mathop{\mathrm{Cov}}(X_1,X_2)^2}{\mathop{\mathrm{Var}}(X_2)} = \mathop{\mathrm{Cov}}(Y,X_1)$$ $$b_1\mathop{\mathrm{Var}}(X_1)\!\left(1-\frac{\mathop{\mathrm{Cov}}(X_1,X_2)^2}{\mathop{\mathrm{Var}}(X_1)\mathop{\mathrm{Var}}(X_2)}\right)
  = \mathop{\mathrm{Cov}}(Y,X_1)$$ $$b_1\mathop{\mathrm{Var}}(X_1)(1-\rho_{12}^2) = \mathop{\mathrm{Cov}}(Y,X_1)$$ $$b_1 = \frac{\mathop{\mathrm{Cov}}(Y,X_1)}{\mathop{\mathrm{Var}}(X_1)(1-\rho_{12}^2)}
  = \frac{a}{1-\rho_{12}^2}$$

Since $\rho_{12}\neq0$, we have $0 < 1-\rho_{12}^2 < 1$, so: $$\boxed{b_1 = \frac{a}{1-\rho_{12}^2} > a > 0}$$

**Intuition.** $X_2$ is uncorrelated with $Y$ but correlated with $X_1$. When $X_2$ is excluded, part of its correlation with $Y$ (which is zero on net, but the correlation path goes through $X_1$) "leaks" into $a$. Including $X_2$ in the model forces $b_2\neq0$, and the model repartitions the explanatory credit. The coefficient on $X_1$ must increase to compensate for $X_2$ drawing some of the shared variation with $X_1$ away.

More precisely: $b_2 = -b_1\rho_{12}\sqrt{\mathop{\mathrm{Var}}(X_1)/\mathop{\mathrm{Var}}(X_2)} < 0$ (if $\rho_{12}>0$). Adding a negatively-loaded $X_2$ makes $X_1$'s coefficient larger to maintain the same predicted values.

**Key takeaway:** When adding an uncorrelated-with-$Y$ but correlated-with-$X_1$ variable, the coefficient on $X_1$ is inflated by the factor $1/(1-\rho_{12}^2)$. This is the VIF again --- the same mechanism that inflates variance also inflates coefficients.
:::

::: {.freeproblem number="4" difficulty="hard" problemId="quant_interview_fundamental_qdsi_p88"}
**When Does Sequential OLS Equal Multiple Regression?** Step 1: Regress $Y$ on $X_1$; get coefficient $b_1$ and residual $R_Y = Y - b_1 X_1$. Step 2: Regress $R_Y$ on $X_2$; get coefficient $b_2$. Separately, regress $Y$ on both $(X_1,X_2)$ simultaneously to get $(a_1,a_2)$. When is $(a_1,a_2)=(b_1,b_2)$?
:::

::: solutionbox
**Answer:** $(a_1,a_2)=(b_1,b_2)$ if and only if $\mathop{\mathrm{Cov}}(X_1,X_2)=0$ (i.e. $X_1$ and $X_2$ are orthogonal).

**The Frisch--Waugh--Lovell (FWL) Theorem.** The FWL theorem states that the coefficient on $X_2$ from the multiple regression equals the coefficient from regressing the *residual of $Y$ after projecting out $X_1$* on the *residual of $X_2$ after projecting out $X_1$*: $$a_2 = \frac{\mathop{\mathrm{Cov}}(\tilde{Y},\tilde{X}_2)}{\mathop{\mathrm{Var}}(\tilde{X}_2)}$$ where $\tilde{Y}=M_1 Y$, $\tilde{X}_2=M_1 X_2$, and $M_1$ is the annihilator matrix for $X_1$: $M_1 = I - X_1(X_1^\top X_1)^{-1}X_1^\top$.

**Sequential step 2 uses $R_Y$ but not $\tilde{X}_2$.** In step 2, you regress $R_Y = \tilde{Y}$ on $X_2$ (not $\tilde{X}_2$): $$b_2 = \frac{\mathop{\mathrm{Cov}}(\tilde{Y}, X_2)}{\mathop{\mathrm{Var}}(X_2)}$$ This equals $a_2$ iff $\tilde{X}_2 = X_2$, i.e. $M_1 X_2 = X_2$, which means $X_1^\top X_2=0$ --- orthogonality.

Similarly, step 1 gives $b_1 = \mathop{\mathrm{Cov}}(Y,X_1)/\mathop{\mathrm{Var}}(X_1)$, which equals $a_1$ iff $X_2$ adds no information to the coefficient on $X_1$, again requiring $\mathop{\mathrm{Cov}}(X_1,X_2)=0$ (from the T17 P5 OVB formula).

**In summary:**

**FWL applied correctly.** To always recover $(a_1,a_2)$ sequentially:

1.  Regress $Y$ on $X_1$: get $b_1$ and $\tilde{Y}=R_Y$.

2.  Regress $X_2$ on $X_1$: get $\tilde{X}_2 = R_{X_2}$.

3.  Regress $\tilde{Y}$ on $\tilde{X}_2$: get $a_2$. (*Not* on $X_2$!)

And $a_1\neq b_1$ in general; to recover $a_1$, use the full multiple regression.

**Key takeaway:** Sequential OLS equals multiple OLS only when $X_1\perp X_2$. The correct partial-out procedure (FWL) requires residualising *both* $Y$ and $X_2$ against $X_1$.
:::

# Geometry, WLS, and Regularisation

::: {.premiumproblem number="5" difficulty="hard" problemId="citadel_qdsi_p8"}
**Does Geometric Symmetry Imply Regression Slope 1?** $(X,Y)$ is uniform on the rectangle with vertices $(1,2)$, $(2,1)$, $(-1,-2)$, $(-2,-1)$. Run the population regression $Y\approx\alpha+\beta X$. Is $\beta=1$?
:::

::: solutionbox
**No.** Despite the rectangle's long axis having slope 1, the regression slope is $\beta = 4/5 \neq 1$.

**Setup.** The four vertices form a parallelogram centred at the origin. Parameterise: $(X,Y) = (1,2) + s(1,-1) + t(-3,-3)$ for $s,t\in[0,1]$, giving $X = 1+s-3t$ and $Y = 2-s-3t$.

Since the distribution is symmetric under $(X,Y)\mapsto(-X,-Y)$: $\mathbb{E}[X]=\mathbb{E}[Y]=0$, so $\alpha=0$.

**Compute moments.** $$\mathbb{E}[X^2] = \int_0^1\!\!\int_0^1(1+s-3t)^2\,ds\,dt = \frac{5}{6}$$ $$\mathbb{E}[XY] = \int_0^1\!\!\int_0^1(1+s-3t)(2-s-3t)\,ds\,dt = \frac{2}{3}$$

**Regression slope.** $$\beta = \frac{\mathbb{E}[XY]}{\mathbb{E}[X^2]} = \frac{2/3}{5/6} = \frac{2}{3}\cdot\frac{6}{5}
  = \boxed{\frac{4}{5}}$$

**Why is $\beta\neq1$?** The long axis of the rectangle has slope 1, but the regression slope is $\rho(X,Y)$ when $\mathop{\mathrm{Var}}(X)=\mathop{\mathrm{Var}}(Y)$ (which holds here since $\mathbb{E}[X^2]=\mathbb{E}[Y^2]=5/6$). Since $\rho = \mathbb{E}[XY]/\mathbb{E}[X^2] = 4/5 < 1$, the slope is strictly less than 1.

The rectangle has significant width perpendicular to the long axis (the short sides have slope $-1$, spreading the distribution in the $X+Y$ direction). This perpendicular spread reduces the correlation from 1 to $4/5$.

**Intuition: regression slope $\neq$ slope of the main axis.** The regression slope $\beta = \rho\cdot\sigma_Y/\sigma_X = \rho$ (here $\sigma_X=\sigma_Y$). The principal axis has slope 1 (corresponding to the eigenvector of the covariance matrix), but the regression line is the conditional mean $\mathbb{E}[Y\mid X]$, which is pulled toward the overall mean (0) by the within-column spread.

**Key takeaway:** Regression slope $=\rho$ when $\mathop{\mathrm{Var}}(X)=\mathop{\mathrm{Var}}(Y)$. Geometric symmetry (the axis of the shape has slope 1) does not imply regression slope 1. The regression line is always "flatter" than the principal axis due to regression to the mean.
:::

::: {.premiumproblem number="6" difficulty="medium" problemId="sig_qdsi_p11"}
**Weighted Regression Under Heterogeneous Data Density.** In linear regression, given data points with different densities at different areas, what is the best fitting strategy? When does WLS reduce to OLS?
:::

::: solutionbox
**Weighted Least Squares (WLS).** When observations have heterogeneous precision, WLS minimises the *weighted* sum of squared residuals: $$\hat{\beta}_{\text{WLS}} = \arg\min_\beta \sum_{i=1}^n w_i(y_i-x_i^\top\beta)^2
  = (X^\top W X)^{-1}X^\top W y$$ where $W = \mathrm{diag}(w_1,\ldots,w_n)$ and $w_i > 0$.

**When to use WLS.**

*Case 1: Heteroskedasticity.* If $\mathop{\mathrm{Var}}(\varepsilon_i)=\sigma^2/w_i$ (varying noise), set $w_i$ proportional to precision ($=1/\mathop{\mathrm{Var}}$): $$w_i = \frac{1}{\mathop{\mathrm{Var}}(\varepsilon_i)}$$ This makes WLS the GLS (Generalised Least Squares) estimator, which is BLUE when errors have this structure.

*Case 2: Density-weighted regression.* If dense regions of the data space are over-represented and you want the model to fit sparse regions equally well, down-weight high-density regions: $$w_i \propto \frac{1}{f(x_i)}$$ where $f$ is the empirical density. This produces a model that minimises the *expected squared error over the covariate distribution* rather than over the sample distribution.

*Case 3: Aggregated data.* If observation $i$ is an average of $n_i$ raw data points, set $w_i = n_i$ (weighted by group size).

**When WLS reduces to OLS.** WLS $=$ OLS when all weights are equal: $w_1=w_2=\cdots=w_n=c$. In matrix form: $W = cI$, so $(X^\top WX)^{-1}X^\top Wy
= c(X^\top X)^{-1}X^\top y/c = (X^\top X)^{-1}X^\top y = \hat{\beta}_{\text{OLS}}$.

**Key takeaway:** WLS is OLS with a re-weighting of observations. Set $w_i=1/\sigma_i^2$ to correct for heteroskedasticity, or $w_i\propto1/f(x_i)$ to balance data density. WLS reduces to OLS when all weights are equal (homoskedastic, uniform-density case).
:::

::: {.premiumproblem number="7" difficulty="hard" problemId="quant_interview_fundamental_qdsi_p117"}
**Ridge vs Lasso: Bias--Variance Trade-off.**

1.  Write the optimisation problems for Ridge (L2) and Lasso (L1).

2.  Derive the closed-form Ridge solution.

3.  Why does Lasso produce sparse solutions while Ridge does not?

4.  As $\lambda\to\infty$, what happens to the coefficients?
:::

::: solutionbox
**(a) Optimisation problems.** $$\hat{\beta}_{\text{Ridge}} = \arg\min_\beta
  \underbrace{\|y-X\beta\|_2^2}_{\text{fit}}
  + \lambda\underbrace{\|\beta\|_2^2}_{\text{L2 penalty}}$$ $$\hat{\beta}_{\text{Lasso}} = \arg\min_\beta
  \|y-X\beta\|_2^2 + \lambda\|\beta\|_1
  = \arg\min_\beta \|y-X\beta\|_2^2 + \lambda\sum_j|\beta_j|$$

**(b) Ridge closed-form solution.** Take the gradient of the Ridge objective and set to zero: $$\frac{\partial}{\partial\beta}\left[\|y-X\beta\|_2^2+\lambda\|\beta\|_2^2\right]
  = -2X^\top(y-X\beta) + 2\lambda\beta = 0$$ $$X^\top X\beta + \lambda\beta = X^\top y$$ $$(X^\top X + \lambda I)\beta = X^\top y$$ $$\boxed{\hat{\beta}_{\text{Ridge}} = (X^\top X + \lambda I)^{-1}X^\top y}$$ Note: $X^\top X + \lambda I$ is always invertible for $\lambda>0$, since $X^\top X$ is PSD and $\lambda I$ adds $\lambda>0$ to all eigenvalues.

**(c) Why Lasso produces sparse solutions.**

*Geometric argument.* The Lasso constraint $\|\beta\|_1\leq t$ defines an $\ell^1$ ball (a "diamond" in 2D, an octahedron in 3D) with corners at the coordinate axes. When the OLS contours (ellipsoids) first touch this ball, they are most likely to touch at a corner, where at least one coordinate is exactly zero. The $\ell^2$ ball (Ridge) has no corners, so the contours touch the smooth boundary and no coefficient is exactly zero.

*KKT argument.* The Lasso KKT stationarity condition is: $-2X_j^\top r + \lambda\,\mathrm{sign}(\hat{\beta}_j) = 0$ (if $\hat{\beta}_j\neq0$). For $\hat{\beta}_j=0$: $|-2X_j^\top r|\leq\lambda$ (subgradient). When $\lambda$ is large enough that $|2X_j^\top r|\leq\lambda$, the solution is $\hat{\beta}_j=0$ exactly.

Ridge stationarity: $-2X_j^\top r + 2\lambda\hat{\beta}_j=0
\Rightarrow \hat{\beta}_j = X_j^\top r/\lambda \neq 0$ in general.

**(d) As $\lambda\to\infty$.**

- **Ridge:** $\hat{\beta}_{\text{Ridge}} = (X^\top X+\lambda I)^{-1}X^\top y
      \to \frac{1}{\lambda}X^\top y \to \mathbf{0}$. All coefficients shrink smoothly toward zero.

- **Lasso:** As $\lambda$ increases, coefficients are set to zero one by one (soft-thresholding). For large enough $\lambda$, all coefficients are exactly zero: $\hat{\beta}_{\text{Lasso}}\to\mathbf{0}$.

**Key takeaway:** Ridge shrinks all coefficients but never to exactly zero; Lasso performs variable selection (exact zeros) because the $\ell^1$ penalty's non-differentiability at zero enables the subdifferential condition to be satisfied with $\hat{\beta}_j=0$.
:::

# Technique Summary

::: {.techniquesummary title="Topic 18 Technique Toolkit"}

| Technique | Description |
|---|---|
| **Duplicate data** | $\hat{\beta}$ unchanged; SE $/\sqrt{2}$; $t\times\sqrt{2}$; $R^2$ unchanged; $\chi^2$ doubled. |
| **Combining weak predictors** | Uncorrelated weak predictors can combine to high $R^2$. $R^2_{\text{multi}}\geq R^2_{\text{single}}$ always. |
| **Correlated covariate effect** | Adding $X_2$ with $\mathop{\mathrm{Corr}}(Y,X_2)=0$ and $\mathop{\mathrm{Corr}}(X_1,X_2)\neq0$: coefficient on $X_1$ inflates by $1/(1-\rho_{12}^2)$. |
| **Frisch--Waugh--Lovell** | Sequential OLS $=$ multiple OLS iff $X_1\perp X_2$. Correct FWL: residualise both $Y$ and $X_2$ on $X_1$. |
| **Regression slope** | $\beta = \rho\cdot\sigma_Y/\sigma_X$. When $\sigma_X=\sigma_Y$: $\beta=\rho$. Main axis slope $\neq\beta$ in general. |
| **WLS** | $(X^\top WX)^{-1}X^\top Wy$. Set $w_i=1/\sigma_i^2$ for heteroskedasticity. WLS $=$ OLS when $W\propto I$. |
| **Ridge** | $(X^\top X+\lambda I)^{-1}X^\top y$. Always invertible; shrinks all coeff. toward 0. |
| **Lasso vs Ridge sparsity** | Lasso: $\ell^1$ ball has corners $\Rightarrow$ exact zeros. Ridge: $\ell^2$ ball is smooth $\Rightarrow$ no exact zeros. |

:::

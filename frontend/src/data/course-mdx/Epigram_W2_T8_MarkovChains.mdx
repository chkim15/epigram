# Introduction: Markov Chains in Quant Finance

A **Markov chain** is a stochastic process where the future depends on the present state alone, not on the history. In quant finance, Markov chains model credit ratings, regime switches, order book states, and market microstructure. In interviews, they appear both directly ("find the stationary distribution") and disguised ("expected time to see pattern HH").

::: {.conceptbox title="Markov Chain Fundamentals"}
A (time-homogeneous, discrete-time) Markov chain on state space $\mathcal{S}$ is defined by the **transition matrix** $P$ with $P_{ij} = \mathbb{P}(X_{t+1}=j \mid X_t=i)$.

**Key properties:**

- **Irreducible:** every state is reachable from every other state. A finite irreducible chain has a *unique* stationary distribution.

- **Aperiodic:** the GCD of return times to state $i$ is 1 for all $i$. Irreducible + aperiodic $\Rightarrow$ $P^n \to \mathbf{1}\pi^\top$ as $n\to\infty$.

- **Stationary distribution** $\pi$: satisfies $\pi P = \pi$ and $\sum_i \pi_i = 1$.

- **Expected return time:** for irreducible chains, $m_i = \mathbb{E}[\text{return time to }i] = 1/\pi_i$.
:::

::: {.conceptbox title="Absorbing Chains and the Fundamental Matrix"}
A state is **absorbing** if $P_{ii}=1$. Order states as transient first, absorbing last: $$P = \begin{pmatrix}Q & R \\ 0 & I\end{pmatrix}$$ **Fundamental matrix:** $N = (I-Q)^{-1}$.

- $N_{ij}$ = expected number of times in transient state $j$ starting from transient state $i$.

- Expected steps until absorption from state $i$: $t_i = (N\mathbf{1})_i$ where $\mathbf{1}$ is the all-ones vector.

- Absorption probabilities into absorbing state $k$: $B = NR$.
:::

# Stationary Distributions

::: {.workedbox number="1" difficulty="easy" problemid="quant_interview_fundamental_qdsi_p102"}
**Weather Chain: Stationary Distribution.** The weather is Sunny (S) or Rainy (R). From Sunny: stay Sunny with prob 0.8. From Rainy: become Sunny with prob 0.4.

1.  Write the transition matrix $P$.

2.  Find the stationary distribution $\pi$.

3.  Starting Sunny, find the expected days until the first Rainy day.
:::

::: solutionbox
**(a) Transition matrix** (rows = current state, cols = next state): $$P = \begin{array}{c|cc} & S & R \\ \hline S & 0.8 & 0.2 \\ R & 0.4 & 0.6 \end{array}$$

**(b) Stationary distribution.** Solve $\pi P = \pi$, $\pi_S + \pi_R = 1$: $$\pi_S = 0.8\pi_S + 0.4\pi_R \implies 0.2\pi_S = 0.4\pi_R
  \implies \pi_S = 2\pi_R$$ With $\pi_S + \pi_R = 1$: $2\pi_R + \pi_R = 1 \Rightarrow \pi_R = 1/3$, $\pi_S = 2/3$. $$\boxed{\pi = (2/3,\; 1/3)}$$ **Interpretation:** In the long run, $2/3$ of days are Sunny and $1/3$ are Rainy.

**(c) Expected first passage time from S to R.** Let $h$ = expected days from Sunny until first Rainy day. Condition on the next day: $$h = 1 + 0.8\,h + 0.4\cdot0
  \implies h - 0.8h = 1
  \implies 0.2h = 1
  \implies h = 5$$ Starting Sunny, the expected wait for the first Rainy day is **5 days**.

**Note:** The expected return time to R from R is $1/\pi_R = 3$ days. Since R is visited on average every 3 days and sunny spells last on average 5 days (geometric with success prob $0.2$), these are consistent.

**Key takeaway:** For a 2-state chain with $P_{SR}=\alpha$ and $P_{RS}=\beta$: $\pi_S = \beta/(\alpha+\beta)$, $\pi_R = \alpha/(\alpha+\beta)$. Expected first passage from $S\to R$ is $1/\alpha$.
:::

::: {.freeproblem number="2" difficulty="medium" problemId="quant_interview_fundamental_qdsi_p103"}
**3-State Markov Chain: Expected Return Time.** States $\{1,2,3\}$ with transition matrix: $$P = \begin{pmatrix}0 & 1 & 0 \\ 1/2 & 0 & 1/2 \\ 0 & 1 & 0\end{pmatrix}$$

1.  Find the stationary distribution.

2.  Find the expected return time to each state.

3.  Is this chain aperiodic? Irreducible?
:::

::: solutionbox
**(a) Stationary distribution.** Solve $\pi P = \pi$: $$\begin{align*}
  \pi_1 &= \tfrac{1}{2}\pi_2 \\
  \pi_2 &= \pi_1 + \pi_3 \\
  \pi_3 &= \tfrac{1}{2}\pi_2
\end{align*}$$ From the first and third equations: $\pi_1 = \pi_3 = \frac{1}{2}\pi_2$. With $\pi_1+\pi_2+\pi_3=1$: $\frac{1}{2}\pi_2 + \pi_2 + \frac{1}{2}\pi_2=1
\Rightarrow 2\pi_2=1 \Rightarrow \pi_2 = 1/2$. $$\boxed{\pi = (1/4,\; 1/2,\; 1/4)}$$

**(b) Expected return times.** $m_i = 1/\pi_i$:

- $m_1 = 4$ (state 1 is visited every 4 steps on average)

- $m_2 = 2$ (state 2 is visited every 2 steps)

- $m_3 = 4$

**(c) Classification.**

- **Irreducible?** Yes: $1\to2\to3$ and $3\to2\to1$, so all states communicate.

- **Aperiodic?** No. From state 1: the only way to return to 1 is $1\to2\to1$ (2 steps) or $1\to2\to3\to2\to1$ (4 steps), all even. So state 1 has period 2. The chain is **periodic with period 2**.

**Consequence:** Despite being periodic, the unique stationary distribution still exists for this irreducible chain, and $m_i = 1/\pi_i$ still holds. However, $P^n$ does *not* converge to $\mathbf{1}\pi^\top$.

**Key takeaway:** Irreducibility guarantees a unique $\pi$ and the return-time formula. Aperiodicity is additionally needed for $P^n\to\mathbf{1}\pi^\top$.
:::

# Absorbing Chains and Gambler's Ruin

::: {.premiumproblem number="3" difficulty="hard" problemId="quant_interview_fundamental_qdsi_p104"}
**Absorbing Markov Chain: Gambler's Ruin via Matrices.** A gambler has \$2. Each step: win \$1 (prob $1/3$) or lose \$1 (prob $2/3$). Stop at \$0 (ruin) or \$5 (success).

1.  Set up the absorbing Markov chain.

2.  Find the probability of reaching \$5 before ruin from \$2.

3.  Find the expected steps until the game ends from \$2.
:::

::: solutionbox
**(a) Setup.** States: $\{0,1,2,3,4,5\}$. Absorbing: $\{0,5\}$. Transient: $\{1,2,3,4\}$.

Transition matrix (transient block $Q$, absorption block $R$): $$Q = \begin{pmatrix}
    0 & 1/3 & 0 & 0 \\
    2/3 & 0 & 1/3 & 0 \\
    0 & 2/3 & 0 & 1/3 \\
    0 & 0 & 2/3 & 0
  \end{pmatrix},\quad
  R = \begin{pmatrix}
    2/3 & 0 \\ 0 & 0 \\ 0 & 0 \\ 0 & 1/3
  \end{pmatrix}$$

**(b) Absorption probability via Gambler's Ruin formula.**

For $p\neq q$, the Gambler's Ruin formula gives the probability of reaching $N$ before 0, starting from $k$, with win prob $p$ per step: $$\mathbb{P}(\text{reach }N\text{ before }0 \mid X_0=k)
  = \frac{1-(q/p)^k}{1-(q/p)^N}$$ Here $p=1/3$, $q=2/3$, $q/p=2$, $k=2$, $N=5$: $$\mathbb{P}(\text{ruin}^c \mid X_0=2)
  = \frac{1-2^2}{1-2^5} = \frac{1-4}{1-32} = \frac{-3}{-31}
  = \frac{3}{31} \approx 9.7\%$$

**(c) Expected duration.**

For the biased Gambler's Ruin, the expected duration from state $k$ is: $$\mathbb{E}[T \mid X_0=k]
  = \frac{N}{q-p}\cdot\frac{1-(q/p)^k}{1-(q/p)^N} - \frac{k}{q-p}$$ With $p=1/3$, $q=2/3$, $q-p=1/3$, $N=5$, $k=2$: $$\mathbb{E}[T\mid X_0=2]
  = 3N\cdot\frac{3}{31} - 3k
  = 15\cdot\frac{3}{31} - 6
  = \frac{45}{31} - 6
  = \frac{45-186}{31} = -\frac{141}{31}$$ This is negative, indicating a sign error. Let us use the matrix approach:

$N = (I-Q)^{-1}$. Computing $I-Q$ and inverting (numerically): $t = N\mathbf{1}$ gives the expected absorption time from each state.

For the biased random walk, the correct formula is: $\mathbb{E}[T\mid X_0=k] = \frac{k}{q-p} - \frac{N}{q-p}\cdot
\frac{1-(q/p)^k}{1-(q/p)^N}$. $$= 3k - 3N\cdot\frac{3}{31}
  = 6 - \frac{45}{31} = \frac{186-45}{31} = \frac{141}{31}
  \approx \boxed{4.55 \text{ steps}}$$

**Key takeaway:** The Gambler's Ruin formula $P_k = (1-(q/p)^k)/(1-(q/p)^N)$ is the go-to for biased random walks. The matrix approach $(I-Q)^{-1}$ is more general but requires computation. With $q>p$ (unfair game), the probability of success is very low and the expected duration is short.
:::

# Pattern Waiting Times

The most common Markov chain interview problems involve waiting for a specific pattern in a sequence of coin flips. The key technique is designing a chain whose states track *progress toward the pattern*.

::: {.techniquebox title="Pattern Waiting Time via Markov Chain"}
To find the expected number of flips until pattern $W$ first appears:

1.  Define states as the length of the longest suffix of the sequence so far that is also a prefix of $W$.

2.  Write the transition equations $e_i = 1 + \sum_j P_{ij}e_j$ where state "done" has $e = 0$.

3.  Solve the linear system.

Alternatively, use the **optional stopping theorem** with a martingale.
:::

::: {.workedbox number="4" difficulty="medium" problemid="quant_interview_fundamental_qdsi_p56"}
**Expected Waiting Time for HH vs HT Patterns.** Flip a fair coin repeatedly.

1.  Expected flips until **HH** first appears.

2.  Expected flips until **HT** first appears.

3.  Without calculation: is $\mathbb{E}[T_{HT}]$ bigger or smaller than $\mathbb{E}[T_{HH}]$? Explain.
:::

::: solutionbox
**(a) Expected time for HH.**

States: $\{0,\text{H},\text{HH}\}$ where 0=start/just saw T, H=just saw H (1 H so far), HH=done.

Let $e_0$ = expected flips from state 0, $e_H$ from state H. $$e_0 = 1 + \tfrac{1}{2}e_H + \tfrac{1}{2}e_0
  \qquad\text{(H: go to H; T: stay at 0)}$$ $$e_H = 1 + \tfrac{1}{2}\cdot0 + \tfrac{1}{2}e_0
  \qquad\text{(H: done; T: go to 0)}$$ From the first equation: $\frac{1}{2}e_0 = 1 + \frac{1}{2}e_H
\Rightarrow e_0 = 2 + e_H$. Substituting into the second: $e_H = 1 + \frac{1}{2}(2+e_H)
= 2 + \frac{1}{2}e_H \Rightarrow e_H = 4$. So $e_0 = 6$.

$$\boxed{\mathbb{E}[T_{HH}] = 6}$$

**(b) Expected time for HT.**

States: $\{0,\text{H},\text{HT}\}$. From 0: H (prob 1/2)$\to$H, T (prob 1/2)$\to$`<!-- -->`{=html}0. From H: T (1/2)$\to$done, H (1/2)$\to$H (stay!). $$e_0 = 1 + \tfrac{1}{2}e_H + \tfrac{1}{2}e_0
  \implies e_0 = 2 + e_H$$ $$e_H = 1 + \tfrac{1}{2}\cdot0 + \tfrac{1}{2}e_H
  \implies \tfrac{1}{2}e_H = 1 \implies e_H = 2$$ So $e_0 = 4$. $$\boxed{\mathbb{E}[T_{HT}] = 4}$$

**(c) Intuitive explanation.**

$\mathbb{E}[T_{HT}] = 4 < \mathbb{E}[T_{HH}] = 6$.

When waiting for HH: if you've just seen H and the next flip is T, you've "wasted" the H and must start completely over from state 0.

When waiting for HT: if you've just seen H and the next flip is H, you haven't wasted anything --- you're still in state H, still one flip away from completing HT. The pattern HT is *self-reinforcing* (a new H while in state H keeps you at state H), while HH is *self-defeating* (a T after H sends you back to the start).

**Key takeaway:** Pattern waiting times depend critically on *overlaps*. HT has no overlap (T cannot start a new HT pattern), while HH overlaps with itself (HH contains H as a prefix). Patterns with larger self-overlap take longer on average.
:::

::: {.freeproblem number="5" difficulty="medium" problemId="quant_interview_fundamental_qdsi_p19"}
**Winning Probability in an Alternating Pattern Game.** Players A and B alternate flipping a fair coin (A starts). If the pattern HT appears (head immediately followed by tail), the player who flips the tail wins. What is $\mathbb{P}(A\text{ wins})$?
:::

::: solutionbox
**States.** Track the last flip and whose turn it is next. Define five states:

- **S**: Start (no flip yet), A's turn.

- **AH**: A just flipped H; B's turn next.

- **BH**: B just flipped H; A's turn next.

- **BT**: last flip was T (by B); B's turn next.

- **AT**: last flip was T (by A); A's turn next.

**Transitions.** The game ends when a T is flipped immediately after an H (forming HT); the tail-flipper wins.

- From **AH** (B flips): T(1/2)$\to$**B wins**; H(1/2)$\to$BH.

- From **BH** (A flips): T(1/2)$\to$**A wins**; H(1/2)$\to$AH.

- From **BT** (B flips): H(1/2)$\to$BH; T(1/2)$\to$AT.

- From **AT** (A flips): H(1/2)$\to$AH; T(1/2)$\to$BT.

- From **S** (A flips): H(1/2)$\to$AH; T(1/2)$\to$BT.

**Equations.** Let $p_X$ = Prob(A wins from state $X$). $$p_{\text{AH}} = \tfrac{1}{2}\cdot0 + \tfrac{1}{2}\,p_{\text{BH}}
  = \tfrac{p_{\text{BH}}}{2}$$ $$p_{\text{BH}} = \tfrac{1}{2}\cdot1 + \tfrac{1}{2}\,p_{\text{AH}}
  = \tfrac{1}{2} + \tfrac{p_{\text{AH}}}{2}$$ Solving: $p_{\text{AH}}=\tfrac{1}{3}$, $p_{\text{BH}}=\tfrac{2}{3}$.

$$p_{\text{BT}} = \tfrac{1}{2}p_{\text{BH}} + \tfrac{1}{2}p_{\text{AT}}
  = \tfrac{1}{3} + \tfrac{p_{\text{AT}}}{2}$$ $$p_{\text{AT}} = \tfrac{1}{2}p_{\text{AH}} + \tfrac{1}{2}p_{\text{BT}}
  = \tfrac{1}{6} + \tfrac{p_{\text{BT}}}{2}$$ Substituting: $p_{\text{BT}} = \tfrac{1}{3}+\tfrac{1}{12}+\tfrac{p_{\text{BT}}}{4}
\Rightarrow \tfrac{3}{4}p_{\text{BT}}=\tfrac{5}{12} \Rightarrow p_{\text{BT}}=\tfrac{5}{9}$.

From Start (A first flips): $$p_S = \tfrac{1}{2}\,p_{\text{AH}} + \tfrac{1}{2}\,p_{\text{BT}}
  = \tfrac{1}{6} + \tfrac{5}{18} = \boxed{\dfrac{4}{9}}$$

**Key takeaway:** Despite A going first, A wins with probability $4/9 < 1/2$. After a T is flipped (by either player), the situation is not symmetric --- it matters whose turn it is next, since only the player who flips T after an H wins. The full 5-state chain is needed to capture this.
:::

::: {.freeproblem number="6" difficulty="medium" problemId="quant_interview_fundamental_qdsi_p9"}
**Expected Waiting Time for Consecutive Heads.** Toss a fair coin repeatedly. Find:

1.  $\mathbb{E}[T_{HH}]$ --- expected flips until HH (two consecutive heads).

2.  $\mathbb{E}[T_{HHH}]$ --- expected flips until HHH.

3.  General formula $\mathbb{E}[T_{H^n}]$ for $n$ consecutive heads.
:::

::: solutionbox
**(a) $\mathbb{E}[T_{HH}] = 6$** from Problem 4(a).

**(b) $\mathbb{E}[T_{HHH}]$.**

States: 0 = (no progress), H = (1 H so far), HH = (2 H so far), done.

Let $e_0, e_1, e_2$ be expected remaining flips from each state: $$\begin{align*}
  e_0 &= 1 + \tfrac{1}{2}e_1 + \tfrac{1}{2}e_0 \implies e_0 = 2 + e_1 \\
  e_1 &= 1 + \tfrac{1}{2}e_2 + \tfrac{1}{2}e_0 \\
  e_2 &= 1 + \tfrac{1}{2}\cdot0 + \tfrac{1}{2}e_0
       \implies e_2 = 1 + \tfrac{e_0}{2}
\end{align*}$$ Substitute $e_2$ into $e_1$: $e_1 = 1 + \frac{1}{2}(1+\frac{e_0}{2}) + \frac{e_0}{2}
= \frac{3}{2} + \frac{3e_0}{4}$. Then $e_0 = 2 + \frac{3}{2} + \frac{3e_0}{4}
\Rightarrow \frac{e_0}{4} = \frac{7}{2}
\Rightarrow e_0 = 14$. $$\boxed{\mathbb{E}[T_{HHH}] = 14}$$

**(c) General formula.**

Let $e_k$ = expected flips from state $k$ (already have $k$ consecutive heads). The recurrence is: $e_k = 1 + \frac{1}{2}e_{k+1} + \frac{1}{2}e_0$ for $k < n$, with $e_n = 0$.

By induction (or by noticing the pattern), solving this system gives: $$\boxed{\mathbb{E}[T_{H^n}] = 2^{n+1} - 2 = 2(2^n - 1)}$$

**Verify:** $n=1$: $\mathbb{E}[T_H] = 2$. $n=2$: $6$. $n=3$: $14$.

**Intuitive derivation:** Use the tower argument. $\mathbb{E}[T_{H^n}] = \mathbb{E}[T_{H^{n-1}}] + \mathbb{E}[\text{additional flips for last H}]$. Given we've just achieved $H^{n-1}$, the next flip is H (prob 1/2, done) or T (prob 1/2, restart from scratch). So: $\mathbb{E}[T_{H^n}] = \mathbb{E}[T_{H^{n-1}}] + 1 + \frac{1}{2}\mathbb{E}[T_{H^n}]$ $\Rightarrow \frac{1}{2}\mathbb{E}[T_{H^n}] = \mathbb{E}[T_{H^{n-1}}] + 1$ $\Rightarrow \mathbb{E}[T_{H^n}] = 2\mathbb{E}[T_{H^{n-1}}] + 2$.

Solving this recurrence with $\mathbb{E}[T_{H^0}]=0$: $\mathbb{E}[T_{H^n}] = 2^{n+1}-2$.
:::

::: {.premiumproblem number="7" difficulty="medium" problemId="quant_interview_fundamental_qdsi_p11"}
**Expected Waiting Time for Consecutive Dice Outcomes.** Roll a fair six-sided die repeatedly. Find the expected number of rolls until you see two consecutive 5's. Generalise to $n$ consecutive 5's.
:::

::: solutionbox
**Expected rolls until two consecutive 5's.**

States: 0 (no 5 just rolled), 1 (last roll was 5), done. Let $e_0, e_1$ be expected rolls from each state. $$e_0 = 1 + \tfrac{1}{6}e_1 + \tfrac{5}{6}e_0
  \implies \tfrac{1}{6}e_0 = 1 + \tfrac{1}{6}e_1
  \implies e_0 = 6 + e_1$$ $$e_1 = 1 + \tfrac{1}{6}\cdot0 + \tfrac{5}{6}e_0$$ Substitute $e_0 = 6+e_1$: $e_1 = 1 + \frac{5}{6}(6+e_1) = 1 + 5 + \frac{5}{6}e_1
\Rightarrow \frac{1}{6}e_1 = 6 \Rightarrow e_1 = 36$. So $e_0 = 42$. $$\boxed{\mathbb{E}[T_{\text{two 5's}}] = 42}$$

**General formula for $n$ consecutive 5's.**

The recurrence mirrors the coin case. Let $p = 1/6$ (prob of 5), $q = 5/6$. Using the same tower argument: $$\mathbb{E}[T_n] = 2\mathbb{E}[T_{n-1}] + \frac{1}{p}
  \quad\text{with}\quad \mathbb{E}[T_0] = 0$$ Wait: let us redo. Given $n-1$ consecutive 5's, the next roll:

- 5 (prob $p$): done. Additional: $1$ roll.

- Not 5 (prob $1-p$): restart from 0. Additional: $1 + \mathbb{E}[T_n]$.

So $\mathbb{E}[T_n] = \mathbb{E}[T_{n-1}] + 1 + (1-p)\mathbb{E}[T_n]
\Rightarrow p\,\mathbb{E}[T_n] = \mathbb{E}[T_{n-1}] + 1
\Rightarrow \mathbb{E}[T_n] = \frac{\mathbb{E}[T_{n-1}]+1}{p}$.

Solving: $\mathbb{E}[T_n] = \frac{1}{p} + \frac{1}{p^2} + \cdots + \frac{1}{p^n}
= \sum_{k=1}^n \frac{1}{p^k}$.

$$\boxed{\mathbb{E}[T_n] = \frac{1}{p}+\frac{1}{p^2}+\cdots+\frac{1}{p^n}
  = \frac{(1/p)^{n+1}-1/p}{1/p-1}
  = \frac{1-p^n}{p^n(1-p)}}$$

**Verify for die ($p=1/6$, $n=2$):** $\mathbb{E}[T_2] = 6 + 36 = 42$. For coin ($p=1/2$, $n=2$): $\mathbb{E}[T_2]=2+4=6$.

**Key takeaway:** The general formula $\mathbb{E}[T_n]=\sum_{k=1}^n p^{-k}$ applies to any i.i.d. sequence with success probability $p$.
:::

::: {.freeproblem number="8" difficulty="medium" problemId="quant_interview_fundamental_qdsi_p6"}
**First Occurrence of HH vs TH in Coin Tossing.** Flip a fair coin until either HH or TH appears as two consecutive flips. If HH appears first you win. Find $\mathbb{P}(\text{you win})$. Can the answer be determined by a symmetry argument?
:::

::: solutionbox
**States.** Track the last flip.

- **S**: Start (no flip yet).

- **H**: last flip was H.

- **T**: last flip was T.

**Transitions.**

- From **S**: H(1/2)$\to$H; T(1/2)$\to$T.

- From **H**: H(1/2)$\to$**HH: you win**; T(1/2)$\to$**TH: you lose**.

- From **T**: H(1/2)$\to$**TH: you lose**; T(1/2)$\to$T (stay).

**Key observation.** State T is a *death trap*: once the last flip is T, the next H completes TH (you lose), and the next T keeps you in state T. You can never win from state T.

**Equations.** Let $p_X$ = Prob(you win from state $X$). $$p_T = \tfrac{1}{2}\cdot0 + \tfrac{1}{2}\,p_T
  \implies p_T = 0$$ $$p_H = \tfrac{1}{2}\cdot1 + \tfrac{1}{2}\cdot0 = \tfrac{1}{2}$$ $$p_S = \tfrac{1}{2}\,p_H + \tfrac{1}{2}\,p_T
  = \tfrac{1}{4} + 0
  = \boxed{\dfrac{1}{4}}$$

**Why the symmetry argument fails.** Both HH and TH end with H, so one might argue the game ends equally on HH or TH. But this ignores that once a T is ever flipped, every future path through state T leads to a loss --- the state T is absorbing into a loss. The only winning path is H on flip 1, H on flip 2 (probability $\frac{1}{2}\times\frac{1}{2}=\frac{1}{4}$).

**Key takeaway:** Despite the symmetric-looking patterns, HH and TH are not equally likely to appear first. Once a T occurs, TH wins with certainty. You win only via the path $S\to H\to\text{HH}$, giving probability $\mathbf{1/4}$.
:::

::: {.premiumproblem number="9" difficulty="hard" problemId="quant_interview_fundamental_qdsi_p7"}
**Which Pattern Appears First in Coin Tossing?** Flip a fair coin repeatedly. Find:

1.  $\mathbb{P}(\text{HHT appears before HTT})$

2.  $\mathbb{P}(\text{HHT appears before HTH})$

3.  $\mathbb{P}(\text{HHH appears before THH})$

4.  $\mathbb{P}(\text{HHT appears before THH})$
:::

::: solutionbox
**Method.** Build a Markov chain whose states are the set of possible "current progress" prefixes toward either pattern. Write transition equations and solve.

**(1) HHT vs HTT.**

States track the longest common suffix that is a prefix of HHT or HTT. The shared prefix H means: after first H, we're at state H for both. From H: next flip H$\to$HH (toward HHT), T$\to$HT (toward HTT). From HH: T$\to$HHT (done, HHT wins); H$\to$HH (stay). From HT: T$\to$HTT (done, HTT wins); H$\to$H (restart).

Let $p_i$ = Prob(HHT wins from state $i$). $$\begin{align*}
  p_{\text{start}} &= \tfrac{1}{2}p_H + \tfrac{1}{2}p_{\text{start}}
  \implies p_{\text{start}} = p_H \\
  p_H &= \tfrac{1}{2}p_{HH} + \tfrac{1}{2}p_{HT} \\
  p_{HH} &= \tfrac{1}{2}\cdot1 + \tfrac{1}{2}p_{HH}
  \implies p_{HH} = 1 \\
  p_{HT} &= \tfrac{1}{2}\cdot0 + \tfrac{1}{2}p_H
\end{align*}$$ From $p_H = \frac{1}{2}\cdot1 + \frac{1}{2}p_{HT}$ and $p_{HT} = \frac{1}{2}p_H$: $p_H = \frac{1}{2} + \frac{1}{4}p_H \Rightarrow \frac{3}{4}p_H = \frac{1}{2}
\Rightarrow p_H = 2/3$.

$\boxed{\mathbb{P}(\text{HHT before HTT}) = 2/3}$

**(2) HHT vs HTH.** Similar analysis (left as exercise): $\mathbb{P}(\text{HHT before HTH}) = \boxed{2/3}$.

**(3) HHH vs THH.**

After the first H: both patterns need HH next. The key difference:

- From state H: next H$\to$HH, next T$\to$start (for HHH), but T$\to$ "T-seen" for THH.

- THH is achieved via T then HH.

States: {start, T, H, TH, HH, THH=done(THH wins), HHH=done(HHH wins)}. Let $p$ = Prob(HHH wins) from start:

From start: H(1/2)$\to$H; T(1/2)$\to$T. From H: H(1/2)$\to$HH; T(1/2)$\to$T. From HH: H(1/2)$\to$HHH(win); T(1/2)$\to$T. From T: H(1/2)$\to$TH; T(1/2)$\to$T. From TH: H(1/2)$\to$THH(lose); T(1/2)$\to$T.

Let $p_s,p_T,p_H,p_{HH},p_{TH}$ be winning probs from each state. $p_{HH} = 1/2$, $p_{TH}=0$. $p_T = \frac{1}{2}p_{TH}+\frac{1}{2}p_T \Rightarrow p_T=0$. $p_H = \frac{1}{2}p_{HH}+\frac{1}{2}p_T = \frac{1}{4}$. $p_s = \frac{1}{2}p_H + \frac{1}{2}p_T = \frac{1}{8}$.

$\boxed{\mathbb{P}(\text{HHH before THH}) = 1/8}$

**(4) HHT vs THH.**

**States:** S (start), H, HH, T, TH. Winning patterns: HHT (we win); THH (we lose).

Key observations:

- From **T** or **TH**: the next H gives TH or THH (we lose); the next T stays in T. So $p_T = p_{TH} = 0$ --- state T is a death trap for us.

- From **HH**: next T $\to$ HHT (win); next H $\to$ HH. So $p_{HH}=1$.

- From **H**: H$\to$HH ($p=1$); T$\to$T ($p=0$). $p_H = \tfrac{1}{2}\cdot1 + \tfrac{1}{2}\cdot0 = \tfrac{1}{2}$.

- From **S**: H$\to$H; T$\to$T. $p_S = \tfrac{1}{2}p_H + \tfrac{1}{2}p_T = \tfrac{1}{4}$.

$\boxed{\mathbb{P}(\text{HHT before THH}) = 1/4}$

Once a T appears before HH is reached, THH wins with certainty. The only winning path is $S\to H\to HH\to\text{HHT}$, with probability $\tfrac{1}{2}\times\tfrac{1}{2}\times1 = \tfrac{1}{4}$.

**Key takeaway:** Pattern-vs-pattern races are highly non-intuitive. The Conway leading number method provides a systematic approach: compute the overlap score for each pattern pair to determine which pattern tends to appear first.
:::

::: {.premiumproblem number="10" difficulty="hard" problemId="quant_interview_fundamental_qdsi_p46"}
**Waiting Time for the Pattern HTH.** Flip a fair coin repeatedly. Find $\mathbb{E}[T_{HTH}]$ --- the expected number of flips until the pattern HTH first appears.
:::

::: solutionbox
**States.** Track the longest suffix of flips so far that is also a prefix of HTH:

- State 0: no progress (or last flip was T after state 0).

- State H: last flip was H (matches prefix "H" of HTH).

- State HT: last two flips were HT (matches prefix "HT").

- State HTH: done.

**Transitions from each state:**

| State | Flip H (prob 1/2) | Flip T (prob 1/2) |
|---|---|---|
| 0 | $\to$ H | $\to$ 0 |
| H | $\to$ H | $\to$ HT |
| HT | $\to$ HTH (done!) | $\to$ 0 |

**Note:** From state H, flipping H keeps you at H (not 0!) because the new H could be the start of a new HTH. From state HT, flipping T goes to 0 (not HT---T is not a prefix of HTH).

**Expected flip equations.** Let $e_0, e_H, e_{HT}$ be expected flips to completion from each state: $$\begin{align*}
  e_0  &= 1 + \tfrac{1}{2}e_H + \tfrac{1}{2}e_0
        \implies e_0 = 2 + e_H \\
  e_H  &= 1 + \tfrac{1}{2}e_H + \tfrac{1}{2}e_{HT}
        \implies \tfrac{1}{2}e_H = 1 + \tfrac{1}{2}e_{HT}
        \implies e_H = 2 + e_{HT} \\
  e_{HT} &= 1 + \tfrac{1}{2}\cdot0 + \tfrac{1}{2}e_0
           \implies e_{HT} = 1 + \tfrac{e_0}{2}
\end{align*}$$

**Solve:** $e_H = 2 + e_{HT} = 2 + 1 + \frac{e_0}{2} = 3 + \frac{e_0}{2}$. $e_0 = 2 + e_H = 2 + 3 + \frac{e_0}{2} = 5 + \frac{e_0}{2}$. $\frac{e_0}{2} = 5 \Rightarrow e_0 = 10$.

$$\boxed{\mathbb{E}[T_{HTH}] = 10}$$

**Key takeaway:** The state transitions for HTH are non-trivial because "H" overlaps with itself as a prefix of HTH. Always redraw the state diagram carefully, especially the self-loops and "partial-match" transitions.
:::

::: {.premiumproblem number="11" difficulty="hard" problemId="jane_street_qdsi_p4"}
**Waiting Time for the Pattern HHHHT.** Flip a fair coin repeatedly. Find $\mathbb{E}[T_{HHHHT}]$ --- the expected number of flips until the pattern HHHHT first appears.
:::

::: solutionbox
**States:** 0, H, HH, HHH, HHHH, done (HHHHT).

**Transitions:**

| State | H (prob 1/2) | T (prob 1/2) |
|---|---|---|
| 0 | $\to$ H | $\to$ 0 |
| H | $\to$ HH | $\to$ 0 |
| HH | $\to$ HHH | $\to$ 0 |
| HHH | $\to$ HHHH | $\to$ 0 |
| HHHH | $\to$ HHHH | $\to$ done |

Note: from state HHHH, flipping H keeps you at HHHH (still four H's). Flipping T completes HHHHT.

**Equations.** Let $e_0, e_1, e_2, e_3, e_4$ for states 0, H, HH, HHH, HHHH: $$\begin{align*}
  e_0 &= 1 + \tfrac{1}{2}e_1 + \tfrac{1}{2}e_0
       \implies e_0 = 2 + e_1 \\
  e_1 &= 1 + \tfrac{1}{2}e_2 + \tfrac{1}{2}e_0 \\
  e_2 &= 1 + \tfrac{1}{2}e_3 + \tfrac{1}{2}e_0 \\
  e_3 &= 1 + \tfrac{1}{2}e_4 + \tfrac{1}{2}e_0 \\
  e_4 &= 1 + \tfrac{1}{2}e_4 + \tfrac{1}{2}\cdot0
       \implies \tfrac{1}{2}e_4 = 1 \implies e_4 = 2
\end{align*}$$

**Back-substitute:** $e_3 = 1 + 1 + \frac{e_0}{2} = 2 + \frac{e_0}{2}$. $e_2 = 1 + \frac{1}{2}(2+\frac{e_0}{2}) + \frac{e_0}{2}
= 2 + \frac{3e_0}{4}$. $e_1 = 1 + \frac{1}{2}(2+\frac{3e_0}{4}) + \frac{e_0}{2}
= 2 + \frac{7e_0}{8}$. $e_0 = 2 + e_1 = 4 + \frac{7e_0}{8}
\Rightarrow \frac{e_0}{8} = 4 \Rightarrow e_0 = 32$.

But wait: state HHHH goes back to HHHH on H, so the four H's must be accumulated without restart except on T. Let me recheck $e_4 = 2$ and chain back.

$e_3 = 1 + \frac{1}{2}\cdot2 + \frac{1}{2}e_0 = 2 + \frac{e_0}{2}$.

Going back: $e_0 = 2+e_1 = 2+(1+\frac{e_2}{2}+\frac{e_0}{2})$. Using $e_0 = 2 + e_1$, $e_1 = 1 + \frac{e_2+e_0}{2}$, $e_2 = 1 + \frac{e_3+e_0}{2}$, $e_3 = 2+\frac{e_0}{2}$:

$e_3 = 2 + \frac{e_0}{2}$. $e_2 = 1 + \frac{1}{2}(2+\frac{e_0}{2}) + \frac{e_0}{2} = 2 + \frac{3e_0}{4}$. $e_1 = 1 + \frac{1}{2}(2+\frac{3e_0}{4}) + \frac{e_0}{2} = 2 + \frac{7e_0}{8}$. $e_0 = 2 + 2 + \frac{7e_0}{8} = 4 + \frac{7e_0}{8}$ $\Rightarrow \frac{e_0}{8} = 4 \Rightarrow e_0 = 32$.

But also account for the $e_4 = 2$ term: $e_3 = 1 + \frac{e_4+e_0}{2} = 1 + 1 + \frac{e_0}{2} = 2 + \frac{e_0}{2}$ .

$$\boxed{\mathbb{E}[T_{HHHHT}] = 32 + 2 = \text{let's recount}}$$

Recounting carefully: $e_0 = 4 + \frac{7e_0}{8}$\... but we haven't included $e_4$ properly in $e_1$'s chain.

$e_1 = 1 + \frac{e_2}{2} + \frac{e_0}{2}$. This is correct. The $e_4$ only appears explicitly in $e_3$'s equation. Tracing: $e_0=32$, $e_3=18$, $e_2=26$, $e_1=30$. Check: $e_0=2+e_1=2+30=32$.

$$\boxed{\mathbb{E}[T_{HHHHT}] = 32}$$

**Pattern using general formula.** For HHHHT (4 heads then 1 tail): The state HHHH is a "restart-absorbing" state that keeps absorbing heads. The answer matches: $\mathbb{E}[T_{H^4T}] = \sum_{k=1}^4 2^k + 2^4\cdot
\mathbb{E}[\text{geometric}(1/2)] = 2+4+8+16 + ...$

Actually by the general consecutive-H formula with the additional tail: $\mathbb{E}[T_{HHHHT}] = \mathbb{E}[T_{HHHH}] \cdot$ (adjusted for tail) $= (2+4+8+16)+16/(1/2) - ...$

The direct calculation gives $e_0=32$.

**Key takeaway:** State HHHH self-loops on H (four accumulated heads act as a holding state), which creates the $e_4=2$ base case and propagates through the back-substitution.
:::

::: {.premiumproblem number="12" difficulty="veryhard" problemId="citadel_qdsi_p23"}
**Meeting Probability for Two Random-Walking Ants on an Octahedron.** Two ants start at opposite vertices of a regular octahedron. Each turn: Ant A moves first to a uniformly random adjacent vertex, then Ant B moves to a uniformly random adjacent vertex. They stop when one ant moves onto the vertex occupied by the other. What is $\mathbb{P}(\text{A moves onto B's
vertex})$?
:::

::: solutionbox
**Octahedron structure.** A regular octahedron has 6 vertices, each of degree 4. Label two opposite poles $N$ (A starts) and $S$ (B starts). The remaining 4 vertices are *equatorial* and lie on a 4-cycle $v$--$w$--$u$--$x$--$v$. Every equatorial vertex is adjacent to both poles and to its two equatorial cycle-neighbours; it is *not* adjacent to the equatorial vertex opposite it on the 4-cycle.

**States** (relative position of A and B after each full turn):

- **Opp**: A and B are at opposite vertices.

- **Adj**: A and B are at adjacent vertices.

The game starts in state **Opp**.

**Let** $p = \mathbb{P}(\text{A catches B first}\mid\text{Opp})$, $q = \mathbb{P}(\text{A catches B first}\mid\text{Adj})$.

**From Opp** (A at $N$, B at $S$).

A moves first: all 4 of $N$'s neighbours are equatorial, none is $S$, so A *cannot* catch B. A lands on some equatorial vertex $v$.

B then moves from $S$: $S$'s 4 neighbours are all equatorial.

- B moves to $v$ (prob $\tfrac{1}{4}$): B lands on A --- B wins.

- B moves to an equatorial neighbour of $v$ on the 4-cycle (prob $\tfrac{2}{4}$): A and B are now adjacent $\to$ state **Adj**.

- B moves to the equatorial vertex opposite $v$ (prob $\tfrac{1}{4}$): A and B are now opposite $\to$ state **Opp**.

$$p = 0\cdot\tfrac{1}{4} + q\cdot\tfrac{2}{4} + p\cdot\tfrac{1}{4}
  \;\implies\; \tfrac{3p}{4} = \tfrac{q}{2}
  \;\implies\; \boxed{p = \tfrac{2q}{3}} \tag{1}$$

**From Adj** (A at equatorial $v$, B at equatorial $w$, with $v$--$w$ an edge).

Label the 4-cycle $v$--$w$--$u$--$x$--$v$, so $v$'s octahedron neighbours are $\{N,S,w,x\}$ and $w$'s are $\{N,S,v,u\}$.

A moves first (4 equally likely destinations):

- **A$\to w$** (prob $\tfrac{1}{4}$): A lands on B --- **A wins**.

- **A$\to N$** (prob $\tfrac{1}{4}$): A at $N$, B at $w$ (adjacent, since $N$ adj to all equatorial). B moves from $w$ to $\{N,S,v,u\}$:

  - $B\to N$ ($\tfrac{1}{4}$): B catches A --- B wins (contrib 0).

  - $B\to S$ ($\tfrac{1}{4}$): A at $N$, B at $S$ $\to$ **Opp**.

  - $B\to v$ ($\tfrac{1}{4}$): A at $N$, B at $v$ $\to$ **Adj**.

  - $B\to u$ ($\tfrac{1}{4}$): A at $N$, B at $u$ $\to$ **Adj**.

  Contribution: $\tfrac{1}{4}(0 + p + q + q) = \tfrac{p+2q}{4}$.

- **A$\to S$** (prob $\tfrac{1}{4}$): symmetric to A$\to N$. Contribution: $\tfrac{p+2q}{4}$.

- **A$\to x$** (prob $\tfrac{1}{4}$): A at $x$, B at $w$. $x$ and $w$ are opposite on the 4-cycle (2 steps apart), hence opposite vertices of the octahedron. B moves from $w$ to $\{N,S,v,u\}$; none of these equals $x$ (B cannot catch A). $x$ is adjacent to $N$, $S$, $v$, $u$ --- all of B's possible destinations --- so the resulting state is **Adj** in all 4 cases. Contribution: $\tfrac{1}{4}(q+q+q+q) = q$.

Assembling (each A-move has weight $\tfrac{1}{4}$): $$q = \tfrac{1}{4}\!\cdot\!1
    + \tfrac{1}{4}\!\cdot\!\tfrac{p+2q}{4}
    + \tfrac{1}{4}\!\cdot\!\tfrac{p+2q}{4}
    + \tfrac{1}{4}\!\cdot\! q
  = \tfrac{1}{4} + \tfrac{p+2q}{8} + \tfrac{q}{4}$$ $$q - \tfrac{q}{4} = \tfrac{1}{4} + \tfrac{p}{8} + \tfrac{q}{4}
  \;\implies\; \tfrac{q}{2} = \tfrac{1}{4} + \tfrac{p}{8}
  \;\implies\; \boxed{q = \tfrac{1}{2} + \tfrac{p}{4}} \tag{2}$$

**Solving (1) and (2).** Substitute $p = \tfrac{2q}{3}$ into (2): $$q = \tfrac{1}{2} + \tfrac{1}{4}\cdot\tfrac{2q}{3}
  = \tfrac{1}{2} + \tfrac{q}{6}
  \;\implies\; \tfrac{5q}{6} = \tfrac{1}{2}
  \;\implies\; q = \tfrac{3}{5},
  \quad p = \tfrac{2}{3}\cdot\tfrac{3}{5} = \tfrac{2}{5}$$

$$\boxed{\mathbb{P}(\text{A catches B first}) = \dfrac{2}{5}}$$

**Sanity checks.** $p = 2/5 < 1/2$: A is at a slight disadvantage because B *reacts* after A's move, so B sometimes catches A before A can reverse and catch B. $q = 3/5 > 1/2$: from Adjacent, A can immediately land on B (prob 1/4), giving A an advantage.

**Key takeaway:** Set up relative-position states (Opp / Adj) and track who catches whom in each half-turn. The key equation from Opp gives $p = 2q/3$; the careful A$\to x$ branch (landing in Opp, but B's moves all yield Adj) is what distinguishes this from a simpler chain.
:::

# Technique Summary

::: {.techniquesummary title="Topic 8 Technique Toolkit"}

| Technique | Description |
|---|---|
| **Stationary distribution** | Solve $\pi P=\pi$, $\sum\pi_i=1$. For 2-state: $\pi_i = p_{ji}/(p_{ij}+p_{ji})$. |
| **Expected return time** | $m_i = 1/\pi_i$ for irreducible chains. |
| **Periodicity** | Period = GCD of return times. Irreducible + period 1 = ergodic. |
| **Absorbing chains** | $N=(I-Q)^{-1}$; absorption probs $B=NR$; expected time $t=N\mathbf{1}$. |
| **Gambler's Ruin (biased)** | $P_k = (1-(q/p)^k)/(1-(q/p)^N)$. Expected duration: $k/(q-p) - N P_k/(q-p)$. |
| **Pattern waiting: states** | States = length of longest suffix that is a prefix of target. Write $e_i = 1 + \sum P_{ij}e_j$ and solve. |
| **HH vs HT** | $\mathbb{E}[T_{HH}]=6$, $\mathbb{E}[T_{HT}]=4$. Self-overlap makes HH slower. |
| **$n$ consecutive (prob $p$)** | $\mathbb{E}[T_n]=\sum_{k=1}^n p^{-k}$. For fair coin: $2^{n+1}-2$. |
| **Pattern race** | Build a combined chain; solve for absorption probability. Conway's method: compare overlap scores. |
| **Graph random walks** | Reduce by symmetry to equivalence classes. Set up hitting-probability equations. |

:::

# Introduction: Beyond Expectation

Expectation tells you the centre; covariance and correlation tell you how two random variables *move together*. In finance, this distinction is everything: a portfolio of two perfectly correlated assets offers no diversification, while two independent assets reduce variance by half. Understanding when correlations are *feasible* is the deeper skill.

::: {.conceptbox title="Core Definitions and Properties"}
**Covariance:** $\mathop{\mathrm{Cov}}(X,Y) = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y] = \mathbb{E}[(X-\mathbb{E}[X])(Y-\mathbb{E}[Y])]$.

**Correlation:** $\mathop{\mathrm{Corr}}(X,Y) = \rho_{XY} = \dfrac{\mathop{\mathrm{Cov}}(X,Y)}{\sqrt{\mathop{\mathrm{Var}}(X)\mathop{\mathrm{Var}}(Y)}}
\in [-1,1]$.

**Bilinearity of Covariance:** $\mathop{\mathrm{Cov}}(aX+bY, Z) = a\mathop{\mathrm{Cov}}(X,Z)+b\mathop{\mathrm{Cov}}(Y,Z)$.

$\mathop{\mathrm{Var}}(X+Y) = \mathop{\mathrm{Var}}(X)+\mathop{\mathrm{Var}}(Y)+2\mathop{\mathrm{Cov}}(X,Y)$.

$\mathop{\mathrm{Var}}\!\left(\sum_i a_i X_i\right) = \sum_i a_i^2\mathop{\mathrm{Var}}(X_i)
+ 2\sum_{i<j}a_i a_j\mathop{\mathrm{Cov}}(X_i,X_j)$.

**Cauchy--Schwarz:** $|\mathop{\mathrm{Cov}}(X,Y)|\leq\sqrt{\mathop{\mathrm{Var}}(X)\mathop{\mathrm{Var}}(Y)}$, so $|\rho_{XY}|\leq1$.

**$\rho=\pm1$:** $Y=aX+b$ almost surely for some $a\neq0$.
:::

::: {.conceptbox title="Covariance Matrix and Positive Semi-Definiteness"}
For a random vector $(X_1,\ldots,X_n)^\top$, the covariance matrix $\Sigma$ has entries $\Sigma_{ij}=\mathop{\mathrm{Cov}}(X_i,X_j)$. The correlation matrix $C$ has $C_{ij}=\mathop{\mathrm{Corr}}(X_i,X_j)$ (diagonal entries all 1).

A covariance or correlation matrix **must be positive semi-definite (PSD)**: $\mathbf{v}^\top\Sigma\mathbf{v}\geq0$ for all $\mathbf{v}\in\mathbb{R}^n$.

Equivalently, all eigenvalues are $\geq0$.

**Test in 2D:** $\begin{pmatrix}1&\rho\\\rho&1\end{pmatrix}$ is PSD iff $|\rho|\leq1$.

**Test in 3D:** $\begin{pmatrix}1&a&c\\a&1&b\\c&b&1\end{pmatrix}$ is PSD iff $1+2abc - a^2-b^2-c^2\geq0$.
:::

# Computing Covariance and Correlation

::: {.workedbox number="1" difficulty="medium" problemid="quant_interview_fundamental_qdsi_p36"}
**Covariance of Face Counts in Repeated Dice Rolls.** Roll a fair die 5 times. Let $X$ = number of rolls equal to 1 and $Y$ = number of rolls equal to 2. Find $\mathop{\mathrm{Cov}}(X,Y)$ and $\mathop{\mathrm{Corr}}(X,Y)$.
:::

::: solutionbox
**Distributions.** $(X,Y,\ldots)$ follows a $\mathrm{Multinomial}(5;\,
\tfrac{1}{6},\tfrac{1}{6},\ldots)$ distribution. $X\sim\mathrm{Binomial}(5,\tfrac{1}{6})$ and $Y\sim\mathrm{Binomial}(5,\tfrac{1}{6})$ marginally.

**Means and variances:** $\mathbb{E}[X]=\mathbb{E}[Y]=5\cdot\tfrac{1}{6}=\tfrac{5}{6}$. $\mathop{\mathrm{Var}}(X)=\mathop{\mathrm{Var}}(Y)=5\cdot\tfrac{1}{6}\cdot\tfrac{5}{6}=\tfrac{25}{36}$.

**$\mathbb{E}[XY]$ via indicator variables.** Write $X=\sum_{i=1}^5 I_i$ and $Y=\sum_{j=1}^5 J_j$ where $I_i=\mathbf{1}[\text{roll }i=1]$ and $J_j=\mathbf{1}[\text{roll }j=2]$. $$\mathbb{E}[XY] = \sum_{i=1}^5\sum_{j=1}^5 \mathbb{E}[I_i J_j]$$ For $i=j$: $\mathbb{E}[I_i J_i]=\mathbb{P}(\text{roll }i=1\text{ and }=2)=0$.

For $i\neq j$: $\mathbb{E}[I_i J_j]=\mathbb{P}(\text{roll }i=1)\cdot
\mathbb{P}(\text{roll }j=2)=\tfrac{1}{6}\cdot\tfrac{1}{6}=\tfrac{1}{36}$.

There are $5^2-5=20$ pairs with $i\neq j$, so: $\mathbb{E}[XY] = 20\cdot\tfrac{1}{36} = \tfrac{20}{36}$.

**Covariance:** $$\mathop{\mathrm{Cov}}(X,Y) = \mathbb{E}[XY]-\mathbb{E}[X]\mathbb{E}[Y]
  = \frac{20}{36} - \frac{5}{6}\cdot\frac{5}{6}
  = \frac{20}{36} - \frac{25}{36} = -\frac{5}{36}$$

**Correlation:** $$\mathop{\mathrm{Corr}}(X,Y) = \frac{-5/36}{25/36} = -\frac{1}{5}$$

**Multinomial shortcut.** For $\mathrm{Multinomial}(n;p_1,\ldots,p_k)$, $\mathop{\mathrm{Cov}}(X_i,X_j)=-np_ip_j$ for $i\neq j$. Here: $-5\cdot\tfrac{1}{6}\cdot
\tfrac{1}{6}=-\tfrac{5}{36}$.

**Intuition:** More 1s means fewer 2s (they compete for the same 5 rolls), so the covariance is *negative*.

**Key takeaway:** For Multinomial components, $\mathop{\mathrm{Cov}}(X_i,X_j)=-np_ip_j$ always negative. Decompose into indicators to derive it from scratch.
:::

::: {.freeproblem number="2" difficulty="medium" problemId="quant_interview_market_finance_qdsi_p6"}
**Constructing Correlated Normal Random Variables.** Construct a normal $X$ and a second normal $Y$ such that $\mathop{\mathrm{Corr}}(X,Y)=\rho$ for a specified $\rho\in[-1,1]$. How does this generalise to a vector of $n$ correlated normals?
:::

::: solutionbox
**Two-variable construction.** Let $X\sim N(0,1)$ and $Z\sim N(0,1)$ with $X\perp\!\!\!\perp Z$. Define: $$Y = \rho X + \sqrt{1-\rho^2}\,Z$$ Then: $Y\sim N(0,1)$ (since it is a linear combination of independent normals with coefficients summing to 1 in squared terms), $\mathop{\mathrm{Cov}}(X,Y) = \rho\mathop{\mathrm{Cov}}(X,X)+\sqrt{1-\rho^2}\mathop{\mathrm{Cov}}(X,Z) = \rho\cdot1+0=\rho$, and $\mathop{\mathrm{Corr}}(X,Y)=\rho$.

**$n$-variable generalisation via Cholesky decomposition.** Given a target correlation matrix $C$ (which must be PSD), factorise: $C = LL^\top$ where $L$ is lower-triangular (Cholesky factor).

Then if $\mathbf{Z}=(Z_1,\ldots,Z_n)^\top$ with $Z_i\stackrel{\text{iid}}{\sim}
N(0,1)$, the vector $\mathbf{X}=L\mathbf{Z}$ has covariance matrix $\mathbb{E}[\mathbf{X}\mathbf{X}^\top]=L\mathbb{E}[\mathbf{Z}\mathbf{Z}^\top]L^\top=LIL^\top=C$.

**Example: $n=2$.** $C=\begin{pmatrix}1&\rho\\\rho&1\end{pmatrix}$. Cholesky: $L=\begin{pmatrix}1&0\\\rho&\sqrt{1-\rho^2}\end{pmatrix}$. Then $X_1=Z_1$ and $X_2=\rho Z_1+\sqrt{1-\rho^2}Z_2$. Matches the two-variable formula above.

**Key takeaway:** The decomposition $Y=\rho X+\sqrt{1-\rho^2}Z$ is the two-variable Cholesky. For $n$ variables, Cholesky of the correlation matrix produces the construction. This is the standard method for Monte Carlo simulation of correlated assets.
:::

::: {.freeproblem number="3" difficulty="medium" problemId="quant_interview_market_finance_qdsi_p10"}
**Time-Scale Dependent Correlation Paradox.** Can two stocks be positively correlated day-by-day yet negatively correlated over a year? Provide a concrete example or proof.
:::

::: solutionbox
**Yes.** This is possible and is a real phenomenon in financial markets (it relates to the difference between short-run co-movement and long-run mean-reversion).

**Concrete construction.** Let $\varepsilon_t, \eta_t \stackrel{\text{iid}}{\sim} N(0,1)$ for $t=1,\ldots,T$ and let $\mu_t$ be a common factor. Define: $$X_t = \mu_t + \varepsilon_t, \qquad Y_t = -\mu_t + \eta_t + cX_{t-1}$$ with a suitable choice of $c$. A simpler example:

Let $r_{X,t}=Z_t+\varepsilon_t$ and $r_{Y,t}=Z_t+\eta_t$ be daily returns of stocks $X$ and $Y$, where $Z_t$ is a common daily factor and $\varepsilon_t,\eta_t$ are independent noise. Clearly the *daily* correlation is positive: $\mathop{\mathrm{Corr}}(r_{X,t},r_{Y,t})=\mathop{\mathrm{Var}}(Z_t)/
(\mathop{\mathrm{Var}}(Z_t)+\sigma_\varepsilon^2)>0$.

Now suppose the log-price of $X$ is mean-reverting to the log-price of $Y$ over long horizons: $$P_{X,T} = \sum_{t=1}^T r_{X,t},\quad P_{Y,T} = \sum_{t=1}^T r_{Y,t}$$ Add a long-run correction term: $r_{X,t} = Z_t + \varepsilon_t
- \alpha(P_{X,t-1}-P_{Y,t-1})$ with $\alpha>0$. This creates mean-reversion (when $X$ gets far above $Y$ it tends to fall back), making annual cumulative returns *negatively* correlated even though daily returns are positively correlated.

**Minimal clean example.** Two-period model: Day 1: $(X_1,Y_1)$ have correlation $\rho_1>0$ (positive shock). Day 2: each corrects in the *opposite* direction with magnitude $M$. If $M$ is large enough, the annual cumulative returns $(X_1+X_2, Y_1+Y_2)$ become negatively correlated.

**Key takeaway:** Correlation depends on the time horizon. Positive short-run co-movement can be dominated by long-run mean reversion, flipping the sign of annual correlation. This is the basis of *pairs trading*: two stocks that positively co-move daily but mean-revert annually can be traded on their annual spread.
:::

::: {.freeproblem number="4" difficulty="hard" problemId="quant_interview_fundamental_qdsi_p100"}
**Correlation Bounds for Three Random Variables.** Let $X,Y,Z$ be random variables with $\mathop{\mathrm{Corr}}(X,Y)=a$, $\mathop{\mathrm{Corr}}(Y,Z)=b$, $\mathop{\mathrm{Corr}}(X,Z)=c$.

1.  Derive the constraint $(a,b,c)$ must satisfy for the correlation matrix to be positive semi-definite.

2.  Suppose $a=b=0.9$. What is the range of possible values for $c$?
:::

::: solutionbox
**(a) PSD constraint.**

The correlation matrix is: $C = \begin{pmatrix}1&a&c\\a&1&b\\c&b&1\end{pmatrix}$.

For $C$ to be PSD, all principal minors must be $\geq0$ (Sylvester's criterion):

- $1\times1$: $1\geq0$.

- $2\times2$: $1-a^2\geq0$ (and similarly for other pairs). Requires $|a|,|b|,|c|\leq1$.

- $3\times3$ (determinant $\geq0$):

$$\det(C) = 1+2abc - a^2-b^2-c^2 \geq 0$$

**The PSD constraint:** $$\boxed{a^2 + b^2 + c^2 - 2abc \leq 1}$$ with all pairwise $|a|,|b|,|c|\leq1$.

This defines an ellipsoidal region in $(a,b,c)$-space.

**(b) Range of $c$ when $a=b=0.9$.**

Substitute into $\det(C)\geq0$: $$1 + 2(0.9)(0.9)c - (0.9)^2 - (0.9)^2 - c^2 \geq 0$$ $$1 + 1.62c - 0.81 - 0.81 - c^2 \geq 0$$ $$-c^2 + 1.62c - 0.62 \geq 0$$ $$c^2 - 1.62c + 0.62 \leq 0$$ Discriminant: $1.62^2 - 4(0.62) = 2.6244 - 2.48 = 0.1444$. $\sqrt{0.1444}=0.38$. $$c \in \left[\frac{1.62-0.38}{2},\, \frac{1.62+0.38}{2}\right]
  = [0.62,\, 1.00]$$

**Interpretation:** If $X,Y$ and $Y,Z$ are each 90% correlated, then $X,Z$ must also be at least 62% correlated --- transitivity forces high correlation. The intuition: if $X\approx Y$ and $Y\approx Z$, then $X\approx Z$.

**Key takeaway:** The PSD constraint $\det(C)\geq0$ translates to $c\in[ab-\sqrt{(1-a^2)(1-b^2)},\; ab+\sqrt{(1-a^2)(1-b^2)}]$ in general. High pairwise correlations constrain the third correlation tightly.
:::

# Portfolio Theory and Equicorrelation

::: {.premiumproblem number="5" difficulty="hard" problemId="quant_interview_market_finance_qdsi_p2"}
**Maximum Sharpe Ratio Portfolio Construction.** Two assets $A$ and $B$: $\mu_A=1\%$, $\mu_B=2\%$, $\sigma_A=\sigma_B=1\%$, correlation $\rho=0.5$. Construct a portfolio $w=(w_A,1-w_A)$ maximising the Sharpe ratio (assume zero risk-free rate).
:::

::: solutionbox
**Portfolio return and variance.** For weights $(w, 1-w)$: $$\mu_P = w\mu_A + (1-w)\mu_B = w\cdot1\% + (1-w)\cdot2\%
  = 2\% - w\cdot1\%$$ $$\sigma_P^2 = w^2\sigma_A^2 + (1-w)^2\sigma_B^2
  + 2w(1-w)\rho\sigma_A\sigma_B$$ $$= w^2 + (1-w)^2 + 2w(1-w)(0.5) \quad (\text{in units of }\%^2)$$ $$= w^2 + 1 - 2w + w^2 + w(1-w) = 1 - w + w^2 \cdot(2-1) \cdots$$ More carefully: $\sigma_P^2 = w^2 + (1-w)^2 + w(1-w)
= w^2 + 1 - 2w + w^2 + w - w^2 = w^2 - w + 1$.

**Sharpe ratio:** $S(w) = \mu_P/\sigma_P = (2-w)/\sqrt{w^2-w+1}$ (in %).

**Maximise:** Set $dS/dw=0$. Differentiating and setting numerator to zero: $$\frac{d}{dw}\left[\frac{2-w}{\sqrt{w^2-w+1}}\right]
  = \frac{-\sqrt{w^2-w+1}-(2-w)\cdot\frac{2w-1}{2\sqrt{w^2-w+1}}}
         {w^2-w+1} = 0$$ Numerator = 0: $-(w^2-w+1)\cdot2 - (2-w)(2w-1)=0$. $-2w^2+2w-2 - (4w-2-2w^2+w)=0$. $-2w^2+2w-2 - 3w+2+2w^2=0$. $-w=0\Rightarrow w=0$.

Hmm, $w=0$ gives $S=2/1=2$ at 100% B. Check $w=1$: $S=1/1=1$. Check $w=-1$ (short A): $S=3/\sqrt{3}\approx1.73$.

The maximum Sharpe ratio is at $w=0$, i.e., **100% in asset B**.

**Intuition:** Asset $B$ has the higher expected return and the same volatility. With positive correlation, there is no diversification benefit that outweighs simply holding the better asset.

**When does diversification help?** Diversification improves the Sharpe ratio only when $(\mu_A/\sigma_A^2)\neq(\mu_B/\sigma_B^2)$ *and* the correlation is sufficiently low. In this problem both have $\sigma=1\%$ but $\mu_B>\mu_A$, so $B$ dominates on Sharpe.

**Key takeaway:** The tangency (max Sharpe) portfolio weight in asset $A$ is: $$w_A^* = \frac{(\mu_A\sigma_B^2 - \mu_B\rho\sigma_A\sigma_B)}
               {(\mu_A\sigma_B^2 + \mu_B\sigma_A^2
               - (\mu_A+\mu_B)\rho\sigma_A\sigma_B)}$$ When $\sigma_A=\sigma_B$: $w_A^*=(\mu_A-\mu_B\rho)/
((\mu_A+\mu_B)(1-\rho))$. Plugging in: $w_A^*=(1-0.5\cdot2)/(3\cdot0.5)=0$.
:::

::: {.premiumproblem number="6" difficulty="hard" problemId="quant_interview_fundamental_qdsi_p75"}
**Equicorrelation: Feasible Bounds and Construction.** Let $X,Y,Z$ have $\mathop{\mathrm{Corr}}(X,Y)=\mathop{\mathrm{Corr}}(X,Z)=\mathop{\mathrm{Corr}}(Y,Z)=\rho$.

1.  What bounds must $\rho$ satisfy?

2.  Given $\rho$ in the valid range, construct explicit random variables achieving this equicorrelation.

3.  How does this generalise to $n$ variables with all pairwise correlations equal to $\rho$?
:::

::: solutionbox
**(a) Feasibility bounds for $n=3$.**

The equicorrelation matrix is: $C = (1-\rho)I + \rho\mathbf{1}\mathbf{1}^\top$.

Its eigenvalues are $1-\rho$ (with multiplicity $n-1$) and $1+(n-1)\rho$ (with multiplicity 1). For PSD we need all eigenvalues $\geq0$: $$1-\rho\geq0 \implies \rho\leq1
  \qquad\text{and}\qquad
  1+2\rho\geq0 \implies \rho\geq-\tfrac{1}{2}$$ So for $n=3$: $\boxed{\rho\in[-1/2,\,1]}$.

**(b) Construction for any valid $\rho$.**

Let $F\sim N(0,1)$ be a common factor and $\varepsilon_X,\varepsilon_Y,
\varepsilon_Z\stackrel{\text{iid}}{\sim}N(0,1)$ independent of $F$. Define: $$X = \sqrt{\rho}\,F + \sqrt{1-\rho}\,\varepsilon_X
  \quad ({\rho\geq0})$$ Then $\mathop{\mathrm{Var}}(X)=\rho+(1-\rho)=1$ and $\mathop{\mathrm{Cov}}(X,Y)=\rho\mathop{\mathrm{Var}}(F)=\rho$. So $\mathop{\mathrm{Corr}}(X,Y)=\rho$. By symmetry, all three pairwise correlations equal $\rho$.

For $\rho<0$ (down to $-1/2$): use $\sqrt{|\rho|}$ with opposite sign.

**Explicit formula:** $X_i = \sqrt{\rho}\,F + \sqrt{1-\rho}\,\varepsilon_i$ when $\rho\geq0$.

For $\rho\in(-1/(n-1),0)$, a slightly modified construction with multiple factors achieves the target.

**(c) Generalisation to $n$ variables.**

For $n$ equicorrelated variables, the PSD condition on the equicorrelation matrix gives: $$\boxed{\rho \in \left[-\frac{1}{n-1},\; 1\right]}$$ As $n\to\infty$, the lower bound $\to0$: with many variables all mutually negatively correlated, the constraint becomes increasingly tight (you cannot have all pairs strongly negatively correlated).

The one-factor construction $X_i=\sqrt{\rho}F+\sqrt{1-\rho}\varepsilon_i$ achieves any $\rho\in[0,1]$. For negative $\rho$ down to $-1/(n-1)$, a direct construction uses: $X_i = -\frac{1}{\sqrt{n-1}}F + \sqrt{1+\frac{1}{n-1}}\varepsilon_i$ with appropriate scaling.

**Key takeaway:** The equicorrelation lower bound $-1/(n-1)$ is one of the most important constraints in multi-asset correlation modelling. As $n$ grows, correlations between many assets must approach zero from below.
:::

::: {.premiumproblem number="7" difficulty="hard" problemId="quant_interview_fundamental_qdsi_p101"}
**Covariance of Order Statistics.** Let $X_{(1)}\leq X_{(2)}\leq\cdots\leq X_{(n)}$ be the order statistics of $n$ i.i.d. $\mathrm{Uniform}(0,1)$ random variables.

1.  Find $\mathbb{E}[X_{(k)}]$ for the $k$-th order statistic.

2.  Find $\mathop{\mathrm{Cov}}(X_{(1)}, X_{(n)})$ --- the covariance of the minimum and maximum.

3.  Are $X_{(1)}$ and $X_{(n)}$ positively or negatively correlated? Explain intuitively.
:::

::: solutionbox
**(a)** From Topic 3, $\mathbb{E}[X_{(k)}]=k/(n+1)$.

Proof: the joint PDF of $(X_{(1)},\ldots,X_{(n)})$ is $n!$ on the simplex $0<x_1<\cdots<x_n<1$. The marginal of $X_{(k)}$ is: $f_{X_{(k)}}(x)=\frac{n!}{(k-1)!(n-k)!}x^{k-1}(1-x)^{n-k}$, a $\mathrm{Beta}(k,n-k+1)$ distribution. $\mathbb{E}[X_{(k)}]=k/(n+1)$.

**(b)** We need $\mathop{\mathrm{Cov}}(X_{(1)},X_{(n)})=\mathbb{E}[X_{(1)}X_{(n)}]
-\mathbb{E}[X_{(1)}]\mathbb{E}[X_{(n)}]$.

$\mathbb{E}[X_{(1)}]=1/(n+1)$ and $\mathbb{E}[X_{(n)}]=n/(n+1)$.

**Joint PDF of $(X_{(1)},X_{(n)})$.** For $0<u<v<1$: $$f_{X_{(1)},X_{(n)}}(u,v) = n(n-1)(v-u)^{n-2}$$ (Choose which observation is min: $n$ ways; which is max: $n-1$ ways; the remaining $n-2$ observations lie uniformly in $(u,v)$: density $(v-u)^{n-2}$.)

**Compute $\mathbb{E}[X_{(1)}X_{(n)}]$:** $$\mathbb{E}[X_{(1)}X_{(n)}] = n(n-1)\int_0^1\int_0^v uv(v-u)^{n-2}\,du\,dv$$ Let $v$ be fixed, substitute $u=vt$ ($t\in[0,1]$, $du=v\,dt$): $$= n(n-1)\int_0^1 v\cdot v^n\int_0^1 t(1-t)^{n-2}\,dt\,dv
  = n(n-1)\int_0^1 v^{n+1}\,dv\cdot\mathrm{B}(2,n-1)$$ where $\mathrm{B}(2,n-1)=1!/(n-1)!/(n+1-1+1)\ldots$ More carefully: $\int_0^1 t(1-t)^{n-2}dt = \mathrm{B}(2,n-1)=
\frac{1!(n-2)!}{n!}=\frac{1}{n(n-1)}$.

And $\int_0^1 v^{n+1}dv=1/(n+2)$. $$\mathbb{E}[X_{(1)}X_{(n)}] = n(n-1)\cdot\frac{1}{n+2}\cdot\frac{1}{n(n-1)}
  = \frac{1}{n+2}$$

**Covariance:** $$\mathop{\mathrm{Cov}}(X_{(1)},X_{(n)})
  = \frac{1}{n+2} - \frac{1}{n+1}\cdot\frac{n}{n+1}
  = \frac{1}{n+2} - \frac{n}{(n+1)^2}$$ $$= \frac{(n+1)^2 - n(n+2)}{(n+2)(n+1)^2}
  = \frac{n^2+2n+1-n^2-2n}{(n+2)(n+1)^2}
  = \frac{1}{(n+1)^2(n+2)}$$

$$\boxed{\mathop{\mathrm{Cov}}(X_{(1)},X_{(n)}) = \frac{1}{(n+1)^2(n+2)}}$$

**(c)** The covariance is *positive*.

**Intuition:** If the minimum $X_{(1)}$ is large, it pushes all observations upward, including the maximum $X_{(n)}$. Similarly, a small minimum does not prevent the maximum from being large, but a large minimum forces the maximum to be even larger. The two extremes co-vary positively because they are both constrained by the same sample of observations.

**Key takeaway:** Order statistics of uniform i.i.d. data are *positively correlated*, even the most extreme ones. The correlation is: $\mathop{\mathrm{Corr}}(X_{(1)},X_{(n)})=\frac{1/[(n+1)^2(n+2)]}
{1/[(n+1)^2(n+2)]\cdot\ldots}$. For $n=2$: $\mathop{\mathrm{Cov}}=1/12$, $\mathop{\mathrm{Var}}(X_{(1)})=\mathop{\mathrm{Var}}(X_{(2)})=1/12$, so $\mathop{\mathrm{Corr}}=1/2$.
:::
